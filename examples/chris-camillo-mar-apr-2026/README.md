# Reference Example: Chris Camillo, Mar-Apr 2026

This folder contains a fully-worked reference run of the methodology.

## Inputs

`transcripts/` — 3 Dumb Money Live livestream transcripts captured via the [youtube-transcript skill](../../skills/youtube-transcript/SKILL.md):

- `youtube_transcript_1_on_amazon_and_almost_no_one_owns_the_stock_2026-03-12.md` — "#1 on Amazon… and Almost No One Owns the Stock" (1:03:49)
- `youtube_transcript_end_of_war_trades_2026-03-23.md` — "End of War Trades" (1:06:04)
- `youtube_transcript_he_sold_everything_heres_what_hes_buying_2026-04-24.md` — "He Sold Everything... Here's What He's Buying" (1:11:32)

Date range: 2026-03-12 → 2026-04-24 (43-day window).

## Outputs

`outputs/` — 11 files generated by running the methodology against the 3 transcripts:

- `analysis_context.md` — Step 0: context summary, file inventory, speaker-attribution risks
- `raw_evidence_extraction.csv` — Step 1: exhaustive line-by-line raw extraction
- `normalized_mentions.csv` — Step 2: speaker-attributed, ticker-normalized
- `chris_trade_ideas_master.csv` — Step 3: scored master CSV (48 rows, 27 columns)
- `audit_report.md` — Step 4: corrections, uncertainties, speaker-attribution risks
- `chris_trade_ideas_by_episode.md` — Step 5: by-episode breakdown
- `ticker_timeline.md` — Step 5: cross-episode evolution per ticker
- `highest_conviction_ideas.md` — Step 5: top-10 ranked with caveats
- `all_mentions_raw.json` — Step 5: structured JSON of all mentions
- `summary_dashboard.md` — Step 5: overall stats / portfolio / watchlist / avoid lists
- `evidence_traceability.md` — Step 5: every claim mapped to source quote with timestamp

## How this run was produced

The original run was performed manually via Claude Code with the prompt at `prompts/trade-idea-extraction.md` and the creator config at `prompts/creators/chris-camillo.md`.

To reproduce with the automated pipeline:

```bash
python extract.py \
  --transcripts examples/chris-camillo-mar-apr-2026/transcripts/ \
  --creator prompts/creators/chris-camillo.md \
  --output /tmp/chris-camillo-rerun/
```

Then `diff` against `outputs/` to see how the LLM run varies. Some variation across runs is expected — see `evals/extraction_accuracy.md` for how to measure agreement systematically.

## Limitations of this reference dataset

- Sample size: only 3 episodes over 43 days. Conviction-trend analysis is limited.
- Speaker labels NOT present in source transcripts; Chris's contributions inferred from context, first-person language, and known portfolio continuity.
- Transcripts are auto-generated YouTube captions; phonetic transcription errors are common (TSMI→TSM, "air sports"→Amer Sports, etc.).
- Audit confidence: Medium-High overall, with a few flagged uncertainties documented in `audit_report.md`.

This dataset is included for **methodology reproduction and eval development**, not as investment guidance.
